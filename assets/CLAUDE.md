# Claude Code instructions

This repo uses the **minime** agent orchestration flow.
See `ORCHESTRATION.md` for the four phases.
The empirical basis is in `.agent/research/REFERENCES.md`.

Skills are provided by the `minime` Claude Code plugin. Invoke them as
`/minime:plan`, `/minime:implement`, `/minime:review`, `/minime:harvest`.

## Style rules

- **AVOID** the em dash character (\u2014) and separator-style double hyphens written with spaces around them at all costs. Use different sentence structure instead.
